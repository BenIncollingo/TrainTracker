Module: filterTrains

Function allowing filtering of train objects based on search criteria.
Parameters:
Name Type Description
allTrains Array.<object> The list of all train objects available through the Amtrak API.
selectedNumber number The train number to search by.
selectedRoute string The route to search by.
selectedStation string The station to search by.
upcoming boolean Whether to search only for upcoming trains through a station.
fromStation string The departure station to search by.
toStation string The arrival station to search by.
